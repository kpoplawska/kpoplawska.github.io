Oto przykładowy skrypt

1) Skrypt ten sprawdza czy ilość plików wynosi dwa, po czym jeżeli pliki te istnieją sprawdza który z nich ma zawartą większą ilość znaków.

 #!/bin/bash                                             
        echo "Czy liczpa plików wynosi dwa?"                      
 if [ $# -eq 2 ]                 
        then echo "Tak"                      
        if [ -e $1 -a -e $2 ]                   
           x=`cat $1 | wc -m`                                
           y=`cat $2 | wc -m`                        
               then echo                    
                    "Plik $1 posiada wiecej znaków"                          
               else echo "Plik $2 posiada wiecej znaków"
        fi                       
        else echo "Pliki nie istnieja"                             
 fi

2) Ten skrypt natomiast dopisuje do wszystkich plików w katalogu sufiks .old, omijając nasz skrypt.

 #!/bin/bash                                                                                  
 for nazwa in `ls`                                                    
        do 
        if [ $nazwa != `basename $0` ]
        cp $nazwa $nazwa.old 
        done
 exit 0
